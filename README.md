#  Offensive Security Toolkit

A collection of Python tools built while learning cybersecurity, networking, and penetration testing.

This toolkit demonstrates both low-level network programming and the use of industry-standard tools.

---

##  Tools Included

###  Threaded Port Scanner

A custom multi-threaded TCP port scanner built using Python sockets.

**Features:**

* Multi-threaded scanning
* Custom port ranges
* Fast execution

---

###  Banner Grabber

Connects to open ports and retrieves service banners.

**Example:**

```
[+] Port 22 OPEN Banner: SSH-2.0-OpenSSH_6.6.1
[+] Port 80 OPEN Banner: Apache/2.4.7
```

---

###  Advanced Network Scanner

An Nmap-powered scanner for deep network analysis and security assessment.

**Features:**

* Multiple scan modes (Stealth, Standard, Aggressive, Extreme)
* Service and version detection
* OS detection and script scanning
* Risk scoring for detected services
* Domain to IP resolution

---

##  What This Demonstrates

* Understanding of TCP/IP networking
* Multi-threading in Python
* Banner grabbing and service detection
* Integration with tools like Nmap
* Security-focused analysis mindset
